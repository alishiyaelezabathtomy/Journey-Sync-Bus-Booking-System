# Journey Sync - Bus Booking System

## Overview

Journey Sync is a full-stack web-based bus reservation platform developed using Django that allows users to search routes, manage bookings, and store reservation details.

## Objective

The project aims to provide an easy-to-use online bus booking system with efficient backend management and database operations.

## Technologies Used

- Python
- Django
- HTML5
- CSS3
- JavaScript
- SQLite

## Features

- User-friendly interface
- Route search
- Seat booking functionality
- Booking management
- Database storage
- Schedule management

## Backend

Developed using Django framework with SQLite database for managing users, routes, and booking records.

## Future Enhancements

- Online payment integration
- User authentication improvements
- Cloud deployment
